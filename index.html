Natours

This design for a fictional tour agency that provides hiking trips was built with traditional CSS tools. It was coupled with some CSS animations, gradients and the clip-path property to create a nice modern look. This project was also created according to the 7-1 SASS architecture which focuses upon modular development.

Trillo

This design for a fictional travel all-in-one application was built with CSS Flexbox. This tool was used to easily align items in various configuration of the layout. Some advanced CSS animations were used again to make the site feel snappy and modern.

Nexter

This design for a fictional real-estate company was built with the new CSS Grid tool. This makes it a breeze to create complex layouts and making it smoother to create responsive layouts.
